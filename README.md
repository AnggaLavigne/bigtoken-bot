BIGtoken BOT Register With Referral + Verified Email

## Install
      $ pkg update
      $ pkg upgrade
      $ pkg install php
      $ pkg install git
      $ git clone https://github.com/ilyasafr/bigtoken-bot.git
      $ cd bigtoken-bot

## Run
      $ php run.php
